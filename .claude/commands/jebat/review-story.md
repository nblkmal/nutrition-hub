description: 'Review created story and make sure it is aligned'
---

Validate whether the newly create story is valid or not

## Steps

1. Find the newly created story in the stories folder. If story is missing, ask user to point out the story location to be validated.
2. Validate the story follow the story template.
3. Validate the story against the acceptance criteria, project context and coding standards
4. Validate the story have numerical tasks and subtasks.
